# 專案架構

-   src ：存放原始碼和內部標頭檔
-   include ：存放公開標頭檔
-   dist ：存放專案產出
-   bin ：當專案產出為應用程式時，可用 bin 取代 dist
-   test ：存放測試程式
-   example ：存放範例程式
-   doc ：存放說明文件

-   README 或 README.md ：讀我檔案
-   INSTALL ：安裝說明
-   LICENSE ：專案的授權文件
-   HISTORY ：專案的版本釋出記錄
-   .gitignore ：讓 Git 忽略某些形態的目錄或檔案
-   .travis.yml ：Travis CI 的設定檔
-   Makefile ：make(1) 或 gmake(1) 預設的設定檔
-   configure ：Autotools 的命令稿
-   CMakeLists.txt ：cmake(1) 預設的設定檔
